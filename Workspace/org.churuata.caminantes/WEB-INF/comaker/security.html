  • Security measures

Before using a service, observe these safety measures.

  • When an offer is too good to be true. Then that is often the case.
  • Try to check a provider yourself on the internet. For example, search for data about the provider on social media, see if he has a website.
  • Try to get in touch with the provider personally.
  • Search for the physical address on, for example, google maps or better still Streetview to see if it exists and looks reliable.
  • Do not go to the provider alone.
  • Tell friends & family where you are going.
  • Fill in a review afterwards.
